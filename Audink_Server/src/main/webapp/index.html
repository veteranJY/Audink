技术分析


第一部分 情感语音

    基于情感分析的结果计算出各种语音参数，之后利用脱胎于tacotron的CBHG模型助力语音转换，合成出更有感情的声音

第二部分 智能音效

    ①背景音中的情感分析 数据库中会预置大量背景音乐，我们将从节拍，音高变化幅度等方面将其分类打上标签，当需要时对其进行匹配。
    ②匹配合适的背景音效 数据库中会预置大量音效，在第一步中我们已经做好了事件抽取，下一步即根据抽取出来的事件匹配对应的音效。

第三部分 自然语言处理

    基于深度学习的Plutchik情感轮多极情感分析，自我标注数据集三万余句 基于深度学习的配音效判断，利用依存句法分析找到配音关键词
